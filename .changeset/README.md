# Changesets

This folder is managed by [changesets](https://github.com/changesets/changesets).

- Record a change: `pnpm changeset` — pick the bump (patch/minor/major) and write a summary.
- Apply pending changesets to versions + CHANGELOG: `pnpm changeset:version`.
- Publish (if this package is published): `pnpm changeset:release`.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).
